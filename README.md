# HireHunt
An end-to-end data pipeline that collects job postings, cleans and analyzes the data, and presents insights through interactive dashboards.

# The problem 
Every year thousands of job seekers are struggling to analyze the job market. Whether it's finding out which city pays software engineers the most or which skills you need to apply for a certain job or even which companies are hiring the most, you would have to go through hundreds of job postings on different websites and the data is all inconsistent and scattered. That's where HireHunt comes in. HireHunt is an end-to-end data pipeline that collects job posting data and stores that raw data. That raw data is then cleaned and analyzed. The final data is then presented through interactive dashboards.


# How it works
- Data is first collected using an API that gives me raw data
- The data is then stored as raw data so it doesn't have to be re-collected multiple times
- That raw data is then cleaned and analyzed
- Finally the data is presented as a graph or any visualizer for the user
